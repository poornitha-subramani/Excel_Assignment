Excel Basic Data Exploration Assignment
Overview

This project demonstrates basic data exploration and analysis using Microsoft Excel. The assignment includes calculations, conditional analysis, and text manipulation using Excel formulas and functions.

Objectives
Learn basic Excel operations
Apply formulas and functions
Perform data analysis
Understand conditional statements
Practice text functions
Tools Used
Microsoft Excel 2016

Functions and Features Used
Mathematical Functions
SUM()
COUNT()
AVERAGE()
MIN()
MAX()
Conditional Functions
IF()
SUMIF()
COUNTIF()
Text Functions
LEFT()
RIGHT()
MID()
Tasks Performed
Data Calculations

Performed:

Total calculation using SUM()
Counting values using COUNT()
Average calculation using AVERAGE()
Minimum value using MIN()
Maximum value using MAX()
Price Range Column Creation

Created a new column named Price Range using IF condition:

Greater than or equal to 500 → High Price
Less than 500 → Standard Price
Conditional Analysis

Used:

SUMIF() for conditional total calculation
COUNTIF() for conditional counting
Text Functions Columns

Created separate columns using:

LEFT function
RIGHT function
MID function
Learning Outcomes

After completing this assignment, students can:

Use Excel formulas effectively
Perform calculations on datasets
Apply conditional logic
Analyze and summarize data
Extract text using Excel text functions

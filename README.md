Excel Basic Data Exploration Assignment
Overview

This project focuses on basic data exploration and analysis using Microsoft Excel. The assignment was created to understand how Excel functions and formulas can be used to organize, calculate, analyze, and manipulate data efficiently.

The project includes different types of calculations, conditional operations, and text functions that are commonly used in real-world data analysis tasks. Through this assignment, various Excel techniques were practiced to improve spreadsheet handling and analytical skills.

This assignment helps in understanding how Excel can be used for:

Data organization
Data calculation
Conditional analysis
Text manipulation
Basic reporting and exploration
Objectives

The main objectives of this project are:

To learn the fundamentals of Microsoft Excel
To understand how formulas and functions work
To perform mathematical calculations on datasets
To apply conditional logic using Excel formulas
To practice text extraction and manipulation functions
To improve data analysis skills using spreadsheets
To explore data efficiently and generate meaningful outputs
Tools Used
Microsoft Excel 2016
Excel Formulas and Functions
Functions and Features Used
Mathematical Functions

The following mathematical functions were used in this assignment:

SUM()
COUNT()
AVERAGE()
MIN()
MAX()

These functions helped in calculating totals, counting records, finding averages, and identifying minimum and maximum values from the dataset.

Conditional Functions

The following conditional functions were used:

IF()
SUMIF()
COUNTIF()

These functions helped in performing calculations based on specific conditions and criteria.

Text Functions

The following text functions were used:

LEFT()
RIGHT()
MID()

These functions were used to extract characters from text values and create separate columns for text analysis.

Tasks Performed
1. Data Exploration

The dataset was explored and organized properly inside Excel. Different rows and columns were analyzed to understand the structure of the data.

Activities performed:

Checking dataset structure
Organizing columns and rows
Understanding data values
Preparing the dataset for calculations
2. Data Calculations

Basic mathematical calculations were performed using Excel functions.

Calculations Completed
Total values using SUM()
Counting records using COUNT()
Finding averages using AVERAGE()
Finding minimum values using MIN()
Finding maximum values using MAX()

These calculations helped in summarizing and understanding the dataset more effectively.

3. Price Range Column Creation

A new column called Price Range was created using the IF condition.

Logic Used
Greater than or equal to 500 → High Price
Less than 500 → Standard Price

This task helped in understanding conditional statements and category creation in Excel.

4. Conditional Analysis

Conditional functions were applied to analyze data based on specific criteria.

Functions Used
SUMIF() for conditional total calculation
COUNTIF() for conditional counting

These functions helped in filtering and analyzing data based on conditions such as categories, regions, or product types.

5. Text Manipulation

Separate columns were created using text functions.

LEFT Function

Used to extract characters from the beginning of text values.

RIGHT Function

Used to extract characters from the end of text values.

MID Function

Used to extract characters from the middle of text values.

These functions improved understanding of text handling and manipulation in Excel.
